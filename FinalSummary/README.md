# HPI Final API Status Summary

This folder contains the final Excel-only API status summary across all completed HPI evaluation classes.

The workbook intentionally excludes company-level raw data. It summarizes:

- API status by class and tool
- API used / endpoint
- API key status with masked key only
- Reason / notes
- Credits or token evidence
- Latency and record metrics
- Final report inventory

Full API keys remain only in ignored local `.env` files and are not committed.
